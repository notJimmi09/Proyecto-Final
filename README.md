# Nombre del proyecto
FUT-MANAGER
# Descripcion
Este sistema permitirá registrar equipos junto con sus jugadores, almacenar información detallada de cada partido como fecha, lugar y resultados, y generar estadísticas completas de la liga, incluyendo goleadores, asistencias y rendimiento por equipo. Además, ofrecerá la posibilidad de consultar históricos de temporadas pasadas, lo que facilitará el análisis y la comparación de datos a lo largo del tiempo.
# Motivacion
La iniciativa surge de la necesidad de resolver la problemática de pérdida de datos que ocurre cuando la información se maneja en hojas sueltas o archivos dispersos.
Con este proyecto se busca mejorar la organización, reducir errores en cálculos estadísticos y ofrecer transparencia en la gestión de la liga.
